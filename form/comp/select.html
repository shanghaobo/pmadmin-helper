选中交易

选择器选中某一项值时执行的交易。交易说明

选项渲染规则

可配置 sql 来实现动态选项列表。例如：

GetValueFromSQL{select dict_key,dict_value from table_dict where dict_name='test'}

选项变量

选择器的选项绑定的变量，选项数据格式示例为：

[
  {
    "key": "op1",
    "value": "选项1"
  },
  {
    "key": "op2",
    "value": "选项2"
  },
  {
    "key": "op3",
    "value": "选项3"
  }
]

是否显示 key

勾选后选项列表会自动加上 key 前缀，如op1-选项1
